OAuth2 Google Login - Documentation

1. Introduction

This page provides a working implementation of Google OAuth2 login, logout, and user information display. You can log in using your Google account, view user information, and log out securely.

3. Code Examples

HTML, CSS, and JavaScript

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Google Login</title>
</head>
<body>
  <a href="https://go-oauth.vercel.app/auth/?url=http://ndav.me/go-oauth/">Login with Google</a>
  <div id="userInfo"></div>
  <button id="logoutBtn">Logout</button>
  <script>
    const userData = new URLSearchParams(window.location.search).get('user');
    if (userData) {
      localStorage.setItem('userData', decodeURIComponent(userData));
      window.history.replaceState({}, document.title, window.location.origin + window.location.pathname);
    }
    const savedUserData = localStorage.getItem('userData');
    if (savedUserData) {
      const userInfo = JSON.parse(savedUserData);
      document.getElementById("userInfo").innerHTML = `
        <p><strong>Name:</strong> ${userInfo.name}</p>
        <p><strong>Email:</strong> ${userInfo.email}</p>
        <img src="${userInfo.picture}" style="border-radius: 50%; width: 100px;">
      `;
      document.getElementById("logoutBtn").style.display = "block";
    }
    document.getElementById("logoutBtn").addEventListener("click", () => {
      localStorage.removeItem("userData");
      window.location.reload();
    });
  </script>
</body>
</html>

4. API Usage

JavaScript Example

fetch('/auth/google')
  .then(response => response.json())
  .then(data => console.log(data));

Node.js Example

const express = require('express');
const app = express();
app.get('/auth/google', (req, res) => {
  res.redirect('https://go-oauth.vercel.app/auth/?url=http://ndav.me/go-oauth/');
});

Python (Flask) Example

from flask import Flask, redirect
app = Flask(__name__)

@app.route('/auth/google')
def google_login():
    return redirect('https://go-oauth.vercel.app/auth/?url=http://ndav.me/go-oauth/')
