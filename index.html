OAuth2 Google Login - Documentation

1. Introduction

This document provides an overview of how to implement an OAuth2 login using Google. The page allows users to log in with their Google accounts, view their user information, and log out securely.

2. Features

  • Google OAuth2 Login
  • User Information Display (Name, Email, Profile Picture)
  • Logout Functionality

3. Code Overview

Here is a summary of the key parts of the code:

  • Login Button: Redirects to the Google OAuth authentication page.
  • Query Parameter Handling: Extracts user data from the URL after login.
  • LocalStorage: Saves user data for future sessions.
  • Logout Button: Clears the user data from LocalStorage and reloads the page.

4. User Interface

The interface includes the following elements styled with Tailwind CSS and DaisyUI components:

  • Login Button (styled as DaisyUI primary button)
  • User Information Card (name, email, profile picture)
  • Logout Button (styled as DaisyUI secondary button)

5. How to Use

  1. Click the Login with Google button.
  2. Authorize the app to access your Google account.
  3. View your user details displayed on the page.
  4. Click the Logout button to clear your session and reload the page.

6. Technical Notes

Below are some important notes to keep in mind:

  • Ensure the Google OAuth credentials are properly configured.
  • Use HTTPS in production to avoid issues with OAuth redirection.
  • Consider using session storage instead of local storage for better security in production.

7. Security

For improved security, consider the following measures:

  • Use HTTPS to prevent interception of sensitive data.
  • Limit the scope of access requested from Google (e.g., only request profile and email).
  • Use secure cookies instead of localStorage for sensitive information.

8. Sample Code


// Extract user data from URL and store in localStorage
const userData = new URLSearchParams(window.location.search).get('user');
if (userData) {
  localStorage.setItem('userData', decodeURIComponent(userData));
}

// Display user information on the page
const savedUserData = localStorage.getItem('userData');
if (savedUserData) {
  const userInfo = JSON.parse(savedUserData);
  document.getElementById("userInfo").innerHTML = `
    
Name: ${userInfo.name}

    
Email: ${userInfo.email}

    
  `;
}
