OAuth2 Google Login - Documentation

1. Introduction

This page provides a working implementation of Google OAuth2 login, logout, and user information display. You can log in using your Google account, view user information, and log out securely.

2. Try it Out

Login with Google

3. Explanation of the URL

The URL https://go-oauth.vercel.app/auth/?url={YOUR URL BASE APP} is used to redirect the user to the OAuth2 authorization flow. Here’s a breakdown of the URL:

  • https://go-oauth.vercel.app/auth/: The endpoint for initiating the OAuth2 authorization process.
  • url={YOUR URL BASE APP}: This parameter specifies the base URL of your application where the user will be redirected after successful login. Replace {YOUR URL BASE APP} with the actual URL of your app (e.g., https://example.com/dashboard).

4. Code Examples

HTML, CSS, and JavaScript

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Google Login</title>
</head>
<body>
  
  <a href="https://go-oauth.vercel.app/auth/?url={YOUR URL BASE APP}">Login with Google</a>
  
  <div id="userInfo"></div>
  
  <button id="logoutBtn">Logout</button>
  <script>
    // Mengambil data user dari URL dan menyimpannya ke localStorage
    const userData = new URLSearchParams(window.location.search).get('user');
    if (userData) {
      localStorage.setItem('userData', decodeURIComponent(userData));
      window.history.replaceState({}, document.title, window.location.origin + window.location.pathname);
    }
    
    // Menampilkan informasi user yang tersimpan di localStorage
    const savedUserData = localStorage.getItem('userData');
    if (savedUserData) {
      const userInfo = JSON.parse(savedUserData);
      document.getElementById("userInfo").innerHTML = `
        <p><strong>Name:</strong> ${userInfo.name}</p>
        <p><strong>Email:</strong> ${userInfo.email}</p>
        <img src="${userInfo.picture}" style="border-radius: 50%; width: 100px;">
      `;
      document.getElementById("logoutBtn").style.display = "block";
    }
    
    // Logout dan menghapus data user dari localStorage
    document.getElementById("logoutBtn").addEventListener("click", () => {
      localStorage.removeItem("userData");
      window.location.reload();
    });
  </script>
</body>
</html>

Node.js Example

const express = require('express');
const app = express();

app.get('/auth/google', (req, res) => {
  res.redirect('https://go-oauth.vercel.app/auth/?url={YOUR URL BASE APP}');
});

Golang Example

package main

import (
  "net/http"
)

func handler(w http.ResponseWriter, r *http.Request) {
  http.Redirect(w, r, "https://go-oauth.vercel.app/auth/?url={YOUR URL BASE APP}", http.StatusFound)
}

func main() {
  http.HandleFunc("/auth/google", handler)
  http.ListenAndServe(":8080", nil)
}
